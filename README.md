## 🌟 為什麼選擇這個腳本？

如果你也厭倦了 YouTube 越來越像 TikTok（充滿 Shorts）、佈滿廣告彈窗、或總是被演算法推薦一堆沒人看過、品質低落的影片，那麼「YouTube 淨化大師」就是為你量身打造的。

| 😤 使用前 | 😊 使用後 |
|:---:|:---:|
| **注意力被 Shorts 吞噬** | ✅ Shorts 區塊完全消失，找回專注力 |
| **充斥低品質、低觀看影片** | ✅ 自動過濾低門檻影片，內容更精良 |
| **被 Anti-Adblock 彈窗打斷** | ✅ 安靜自動處理，播放不再中斷 |
| **不小心點開影片就跳轉** | ✅ 預設新分頁開啟，瀏覽體驗不中斷 |
| **首頁充滿「合輯」與「雜訊」** | ✅ 介面極度簡約，只保留純粹影片 |

## ✨ 功能亮點

### 🧼 介面極致淨化
- **Shorts 全面封鎖**：徹底移除首頁、搜尋、側邊欄的所有 Shorts 節點。
- **區塊過濾器**：自動隱藏「耳目一新」、「合輯」、「社群貼文」、「電影片段」等干擾區塊。
- **廣告與彈窗**：自動處理 Adblock 警告，移除 Premium 強推橫幅。

### 🖱️ 瀏覽體驗優化
- **背景新分頁**：點擊影片自動在背景新分頁開啟，保持目前列表位置。
- **通知強化**：通知中心點擊影片也支援新分頁開啟。
- **繁簡轉換**：支援繁簡中文通用過濾，設定一個關鍵字，攔截兩種字體。

## 🧠 智慧過濾邏輯 (Smart Filtering)

擔心過濾太強會漏掉想看的？我們設計了精密的**分層機制**：

1. **強規則 (Strong)** 🚫：針對 **Shorts**、**合輯**、**會員專屬影片**。這些內容預設會被強制隱藏，保持介面清爽。
2. **弱規則 (Weak)** 📉：針對 **低觀看數**、**影片時長**、**關鍵字**。這些規則會「聽從」白名單。
3. **分層白名單** 🛡️：
   - **普通白名單**：加入你喜歡的頻道，確保他們的新片不會因為觀看數低而被隱藏。
   - **會員白名單**：**唯一**能讓會員專屬影片出現的路徑，兼顧支持創作者與減少干擾。

## ⚡️ 安裝教學

### 1️⃣ 安裝管理器
請先安裝瀏覽器擴充插件： [**Tampermonkey**](https://www.tampermonkey.net/)。

### 2️⃣ 安裝腳本
點擊下方按鈕即可一鍵安裝：

<div align="center">

[![Install Script](https://img.shields.io/badge/📥_立即安裝_YouTube_淨化大師-安裝腳本-success?style=for-the-badge&logo=tampermonkey)](https://raw.githubusercontent.com/bennytsai1234/YouTube-Cleaner/main/youtube-homepage-cleaner.user.js)
